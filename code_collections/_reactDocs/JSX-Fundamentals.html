JSX is a syntax extension for javascript

Because JSX is a syntax extension for javascript IT IS NOT valid javascript. It must be compiled into valid javascript!

It provides a clear syntax and improved performance for manipulating the DOM

JSX elements are almost identical to HTML.

    Rendering is the act of making the jsx elements appear on screen

    React uses a virtual DOM and a diff tool so that it will only update the DOM when there is a change. So calling render twice between unchanged DOMs will do nothing!

    Assuming there is a div in the html called #app

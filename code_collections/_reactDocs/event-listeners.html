Event Listners are registered right on the elements in JSX

Becuase JSX uses an html like syntax it means that html attributes are used to register events. That is nice because we can see the functions which are bound to events by reading the JSX expressions

React makes sure events have consistent properties across browsers. JSX camelCases the html event attributes it supports. onclick would be onClick in JSX
